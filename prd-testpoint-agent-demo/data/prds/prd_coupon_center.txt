功能：优惠券领取与使用

背景：
用户可以在优惠券中心领取平台发放的优惠券，并在订单结算页选择可用优惠券抵扣订单金额。

功能说明：
1. 用户进入优惠券中心，可以查看可领取、已领取、已使用、已过期的优惠券。
2. 用户点击“立即领取”后，优惠券进入用户卡包。
3. 每个用户对同一张优惠券最多只能领取 1 次。
4. 优惠券有领取开始时间、领取结束时间、使用开始时间和使用结束时间。
5. 订单结算页只展示当前订单可用的优惠券。
6. 使用优惠券支付成功后，优惠券状态变为“已使用”。
7. 订单取消或支付失败时，已锁定的优惠券需要释放回可用状态。

边界条件：
1. 优惠券库存为 0 时不可领取。
2. 到达领取结束时间后不可领取。
3. 到达使用结束时间后不可使用。
4. 订单金额必须大于等于优惠券门槛金额。
5. 优惠券抵扣后订单实付金额不能小于 0 元。

异常场景：
1. 多端同时领取同一张优惠券时，不能超过库存。
2. 用户重复点击领取按钮时，不能重复发券。
3. 支付回调重复到达时，优惠券不能重复核销。
4. 订单支付失败后，优惠券锁定状态必须释放。

预期结果：
系统能够正确完成优惠券领取、展示、锁定、核销、释放和过期处理，并保证库存、用户卡包、订单金额和优惠券状态一致。
